# KMS & Encryption

概要

- AWS Key Management Service (KMS) は暗号鍵管理サービス。

重要ポイント

- 顧客管理のキー（CMK）と AWS 管理のキーの違い
- 暗号化のユースケース（EBS, S3, RDS, Kinesis などとの統合）
- キーポリシーと IAM の関係
- AWS CloudHSM の概要

学習タスク

- KMS キーを作成して S3 バケットと連携する

練習問題

1. データをサーバーサイドで暗号化する際、KMS を使う利点とコストを評価してください。
