# CloudWatch & CloudTrail

概要

- CloudWatch はモニタリングサービス、CloudTrail は API コールの監査用ログサービス。

重要ポイント

- CloudWatch Metrics, Alarms, Logs, Dashboards
- CloudWatch Events / EventBridge
- CloudTrail のログ保存と分析、S3 への配信
- アラート設計と運用フロー

学習タスク

- CloudWatch アラームで自動スケーリングをトリガーする構成
- CloudTrail ログを使った監査演習

練習問題

1. ある操作が行われたときに自動的に通知を受け取りたい。どのサービスをどう組み合わせるか？
